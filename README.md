### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/igaimerca/atlp-capstone-project-server.git
   ```
2. Install packages
   ```sh
   yarn install
   ```
2. Run in DEV mode
   ```sh
   yarn start
   ```
